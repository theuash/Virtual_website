## ⚙️ Before You Start

### Edit `backend/.env` File
This file contains configuration. Update it with your MongoDB details:

```env
# For Local MongoDB:
MONGODB_URI=mongodb://localhost:27017/virtual-platform

# OR For MongoDB Atlas (Cloud):
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/virtual-platform?retryWrites=true&w=majority
```

## ⚡ Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| MongoDB won't connect | Check MONGODB_URI in .env |
| Port 5000 in use | Edit .env, change PORT to 5001 |
| npm install fails | Run: `npm install --legacy-peer-deps` |
| Frontend won't load | Check browser console (F12) for errors |
| API not working | Verify backend server is running |
